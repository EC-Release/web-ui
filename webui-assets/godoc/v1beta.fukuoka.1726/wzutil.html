func Branding ¶ 

func Branding(confmain, plgheader, confheader, authheader, brand, logo, copyright, ca, issue string)

confmain decides the directory to save configuration files. plgheader identifies the plugin head mostly in yaml. confheader specifies the configuration header in yaml. authheader will appear in the HTTP request header for any auth purpose. the "brand" shown as part of the agent logs. the "logo","copyright,"issue" will all display in the product informatin, whereas the ca indicates the issuer's URL and will receive the API calls from the agent
