func Branding ¶ 

func Branding(bc *BrandingConfig)

confmain decides the directory to save configuration files. plgheader identifies the plugin head mostly in yaml. confheader specifies the configuration header in yaml. authheader will appear in the HTTP request header for any auth purpose. subheader serves as subscription header for any service call. instcfheader to indicate the CF instance in the http header instcfenv is to indicate the current CF instance index instheader to indicate the default instance in the header instenv is to show the current instance index by default the "brand" shown as part of the agent logs. the "logo","copyright,"issue" will all display in the product informatin, whereas the ca indicates the issuer's URL and will receive the API calls from the agent
